Class
name description
.sp--toggle 토글의 컨테이너 요소.
.sp--toggle-btn 토글 컨테이너에 영향을 주는 버튼
.sp--toggle-on 토글이 활성화 됐음을 인지시키는 클래스
Attribute
name description
data-target default : 'div' ( string )
버튼의 부모를 찾아서 토글을 수행한다.
없다면 스스로를 토글한다.
data-scope default : '' ( string )
현재는 1개의 옵션만 존재 ( chain )
체인으로 연결된 토글은 부모끼리 서로 상호동작한다.
data-class default : '' ( string )
특정 클래스를 토글하고 싶을때 정의한다.
기본적으로 토글의 명칭은 ".sp--toggle-on" 이다.
만약 .test 라는 클래스를 토글하고 싶다면,
data-class=".test" 라고 정의하자.
Etc

토글의 애니메이션은 설계된 바 없다.
기본적으로 토글은 숨김, 보여줌의 형태가 아닌
활성화, 비활성화의 형태로 마크업 된다. 때문에 컨테이너의 애니메이션은 의미 없다.

토글과 쿠키를 헷갈리지 말자. 완전 다른 기능이다.
