Class
name description
.sp--accordion 아코디언의 컨테이너를 선언
.sp--accordion-array 아코디언 내부 배열을 감싸준다.

탭이 활성화 되면 sp--accordion-active 클래스가 추가된다.

sample
.sp--accordion-btn 아코디언 핸들러 ( 상속버튼 )
.sp--accordion-contents 아코디언 컨텐츠 ( 상속 디비전 )
Attribute
name description
data-scope default : '' ( string )
'' or single
single 의 경우, 아코디언이 한개만 활성화 된다.
data-tabevent default : 'click'
click or mouseenter
아코디언이 활성화 되는 마우스 이벤트 규칙
data-initialtab default : '' ( number )
'' or 아코디언의 인덱스 번호
1을 입력하면 첫번째 아코디언이 활성화 된다.
data-effect default : '' ( string )
'' or 'fade' or 'slide'
아코디언 선택시 보여줄 이펙트, 2가지만 사용된다.
