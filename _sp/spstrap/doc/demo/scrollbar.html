.sp--scrollbar 스크롤바 컨테이너

컨테이너 내부는 한개의 엘리먼트로 묶여 있어야 스타일이 용이하다.

Attribute
name description
data-class default : 'os-theme-dark' ( string )
os-theme-dark, os-theme-light
스크롤바 컨테이너의 테마를 결정한다.
컨테이너가 아닌 스크롤바의 색상이니 주의하자.
data-autohide default : '' ( boolean)
'true' 속성은 마우스가 떠나면 스크롤바가 숨겨진다.
