Attribute
name description
.data-platform default: '' ( string )
youtube, vimeo 를 서비스 할 예정 ( 현재는 youtube만 가능 ) 플랫폼을 지정해주는 영역이다.
.data-controller= default : 'false' ( boolean ) true 는 컨트롤러를 화면에 표시한다.

시작, 중지, 음소거, 프로그래스바 지원

.data-videoid 비디오의 ID를 입력하는 구간이다.

혹시 모르니, url 전체를 입력해도 가져올 수 있도록 개발했다.

.data-loop default: 'false' ( boolean )
true일 경우 비디오가 반복 재생된다.
.data-mute default: 'false' ( boolean )
true일 경우 음소거된다.
.data-volume default: '50' ( number )
볼륨의 크기를 설정한다. ( 숫자만 입력 가능 0 - 100 )
