SPSTRAP
SPSTRAP 이란?
SPSTRAP은 프론트엔드 개발자에게 필요한 css, html, javascript를 통합 관리하도록 제작된 오픈 소스 라이브러리 입니다.
icon은 webfont를 패키지로 묶어서 사용하고 있으며, 소셜 아이콘 11개가 포함된 무료 SVG 아이콘 89종이 제공됩니다.
SVG 아이콘은 flaticon, iconmonstr의 무료 아이콘이 사용되었으며, icomoon으로 패키지화 되어 있습니다.

css 애니메이션, 소셜 아이콘, 드롭다운, 버튼, 박스 레이아웃 등 실무에 많이 사용되는 대부분의 코드가 정리되었으며,
jquery를 사용하지 않고, vanilla javascript ( es6 ) 로 제작되었습니다.

현재까지 총 용량 : 5.75MB (6,034,888 바이트)
( 설명서, 아이콘, 이미지가 모두 포함된 용량이며, 대략 5mb 내외로 예상하자. )
